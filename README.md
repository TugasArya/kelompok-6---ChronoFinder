# kelompok-6---ChronoFinder
ChronoFinder adalah program Java sederhana yang mengimplementasikan Dooms day Algorithm (yang dikembangkan oleh John Horton Conway pada tahun 1973). Program ini berfungsi untuk menentukan hari berdasarkan tanggal dari input user. Dan program ini juga mengintegrasikan konsep konsep if/switch case, looping,  Array, Stack, dan queue
